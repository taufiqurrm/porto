## 📋 Tentang Proyek

Website portfolio pribadi milik **Taufiqur RM** — seorang Guru IT, Web Developer, dan Digital Creator dari Pamekasan, Indonesia. Dibangun dengan HTML, CSS, dan JavaScript murni tanpa framework, namun tetap modern, responsif, dan cepat.

---

## ✨ Fitur

- ⚡ **Pure HTML/CSS/JS** — tanpa framework, ringan dan cepat
- 📱 **Fully Responsive** — tampil sempurna di mobile, tablet, dan desktop
- 🎨 **Modern Design** — UI bersih dengan palet warna biru profesional
- 🔗 **Smooth Scroll Navigation** — navigasi sticky dengan active state otomatis
- 🖼️ **Foto Embedded** — foto dikodekan Base64, tidak perlu file eksternal
- 🌙 **Hover Animations** — interaksi halus pada card dan tombol
